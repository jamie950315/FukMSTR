# Research V85 Commands

Run the BTCUSDC rescue hypothesis closure audit:

```bash
make btcusdc-rescue-closure-v85
```

Run the focused test target:

```bash
make test-btcusdc-v85
```

This audit consolidates V80 through V84 and records whether the existing BTCUSDC route and rescue hypotheses are exhausted. It does not promote a strategy.
